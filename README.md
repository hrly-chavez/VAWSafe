To do List after cloning

1. cd VAWSafe
2. py -m venv venv
3. venv/Scripts/Activate
4. pip install -r requirements.txt
5. cd frontend
6. npm install


How to run project
1. cd backend
2. python manage.py runserver

3. cd frontend
4. npm start
